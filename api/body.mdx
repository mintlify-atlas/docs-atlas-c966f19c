## Overview

The `Body` class is the core component of the Gravity Sim physics engine. Each body represents a physical object affected by gravitational forces from other bodies. Bodies can be dynamic (affected by gravity) or static (fixed in place).

## Methods

### update

<ParamField path="update" type="(dt: number, bodies: Body[]) => void">
  Updates the body's state for a single frame based on gravitational forces from other bodies.
</ParamField>

**Parameters:**

<ParamField path="dt" type="number" required>
  Delta time in milliseconds since the last frame.
</ParamField>

<ParamField path="bodies" type="Body[]" required>
  Array of all bodies in the simulation (including this body).
</ParamField>

**Description:**

The `update` method performs the following steps:
1. Skips update if the body is static
2. Calculates all gravitational forces from other bodies
3. Computes acceleration using F = ma (a = F/m)
4. Updates velocity based on acceleration
5. Updates position based on velocity
6. Calculates current speed
7. Resets net force for the next frame

**Example:**

```typescript
const bodies = [Sun, Earth, Mars];

tc.loop((dt: number) => {
  for (let body of bodies) {
    body.update(dt, bodies);
  }
});
```

### calculateAllForces

<ParamField path="calculateAllForces" type="(bodies: Body[]) => void">
  Calculates the total gravitational force acting on this body from all other bodies.
</ParamField>

**Parameters:**

<ParamField path="bodies" type="Body[]" required>
  Array of all bodies in the simulation.
</ParamField>

**Description:**

Iterates through all bodies, calculates the gravitational force from each using `calculateShellGravityForce`, and accumulates them into the `netForce` property.

### calculateShellGravityForce

<ParamField path="calculateShellGravityForce" type="(body2: Body) => Vector2">
  Calculates the gravitational force vector exerted on this body by another body using the shell theorem.
</ParamField>

**Parameters:**

<ParamField path="body2" type="Body" required>
  The body exerting gravitational force on this body.
</ParamField>

**Returns:** `Vector2` - The force vector.

**Description:**

This method implements the shell theorem for gravitational force:
- If the distance r >= radius of body2: Uses standard Newton's law F = G * m1 * m2 / r²
- If r < radius of body2: Treats body2 as a shell and calculates effective mass based on the volume within distance r

The force is calculated as:
```
forceMagnitude = G * m1 * m2 / r²  (outside)
forceMagnitude = G * m1 * (m2 * (r/R)³) / r²  (inside)
```

### resolveElasticCollision

<ParamField path="resolveElasticCollision" type="(body1: Body, body2: Body) => void">
  Resolves an elastic collision between two bodies using conservation of momentum.
</ParamField>

**Parameters:**

<ParamField path="body1" type="Body" required>
  The first body in the collision.
</ParamField>

<ParamField path="body2" type="Body" required>
  The second body in the collision.
</ParamField>

**Description:**

This method implements elastic collision physics:
1. Checks if bodies are moving toward each other
2. Calculates collision angle
3. Rotates velocity vectors to collision coordinate system
4. Applies 1D conservation of momentum equations
5. Rotates velocities back to world coordinates
6. Updates both bodies' velocities

The collision equations use:
```
v1 = u1 * (m1 - m2) / (m1 + m2) + u2 * 2 * m2 / (m1 + m2)
v2 = u2 * (m2 - m1) / (m1 + m2) + u1 * 2 * m1 / (m1 + m2)
```

### calculateSpeed

<ParamField path="calculateSpeed" type="() => void">
  Calculates and updates the speed property based on the current velocity vector.
</ParamField>

**Description:**

Computes the magnitude of the velocity vector:
```
speed = √(vx² + vy²)
```

## Complete Example

```typescript
import { TinyCanvas } from './drawing/TinyCanvas.js';
import Body from './Body/Body.js';

const tc = new TinyCanvas("canvas");
const bodies: Body[] = [];

// Create a sun
const Sun = new Body({
  position: { x: tc.width / 2, y: tc.height / 2 },
  mass: 1000000 * 10000000,
  radius: 250,
  color: 'orange',
  isStatic: true
});

// Create Earth with initial velocity
const Earth = new Body({
  position: { x: tc.width / 3, y: tc.height / 3 },
  mass: 1000000 * 100000,
  radius: 45,
  color: 'green'
});

bodies.push(Sun, Earth);

// Animation loop
tc.loop((dt: number) => {
  tc.clear();
  
  for (let body of bodies) {
    // Draw the body
    tc.circle(body.position.x, body.position.y, body.radius, body.color);
    
    // Update physics
    body.update(dt, bodies);
  }
  
  // Display Earth's speed
  const speedInCmPerSec = (Earth.speed * 1000 / 36).toFixed(2);
  tc.text(speedInCmPerSec + 'cm/s', 140, 140, 70);
});

// Add interactivity
document.addEventListener('click', () => {
  Earth.velocity.x += 0.3;
});
```
