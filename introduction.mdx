## What is Gravity Sim?

Gravity Sim is a lightweight, browser-based physics engine that simulates gravitational interactions between celestial bodies in real-time. Built entirely in TypeScript, it demonstrates Newtonian gravity with support for both static and dynamic bodies, elastic collisions, and interactive controls.

The simulation uses the shell theorem for accurate gravitational calculations, making it suitable for educational purposes, game development prototypes, or as a foundation for more complex physics simulations.

## Use Cases for Physics Simulations

Gravity Sim can serve as a foundation for various applications:

- **Educational Tools**: Visualize orbital mechanics, gravitational fields, and multi-body interactions for physics education
- **Game Prototyping**: Build space games, planetary systems, or gravity-based puzzles with realistic physics
- **Algorithm Testing**: Experiment with numerical integration methods, collision detection, and force calculations
- **Interactive Demos**: Create engaging web-based demonstrations of gravitational concepts

## Quick Example

Here's how simple it is to create a Sun-Earth system:

```typescript
import { TinyCanvas } from './drawing/TinyCanvas.js'
import Body from './Body/Body.js';

const tc = new TinyCanvas("canvas");
var bodies: Body[] = [];

const Sun = new Body({
  position: { x: tc.width / 2, y: tc.height / 2 },
  mass: 1000000 * 10000000,
  radius: 250,
  color: 'orange',
  isStatic: true
})

const Earth = new Body({
  position: { x: tc.width / 3, y: tc.height / 3 },
  mass: 1000000 * 100000,
  radius: 45,
  color: 'green'
})

bodies.push(Sun, Earth);
```
