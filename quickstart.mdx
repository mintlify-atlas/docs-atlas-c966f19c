## Get the Code

First, clone or download the Gravity Sim repository:

<CodeGroup>
```bash Git Clone
git clone https://github.com/Ghostmaysam1/gravity-sim.git
cd gravity-sim
```

```bash Download ZIP
curl -L https://github.com/Ghostmaysam1/gravity-sim/archive/main.zip -o gravity-sim.zip
unzip gravity-sim.zip
cd gravity-sim-main
```
</CodeGroup>

## Compile TypeScript

The project uses TypeScript with ES2015 modules. Compile the source code to JavaScript:

<Steps>
  <Step title="Install TypeScript (if needed)">
    ```bash
    npm install -g typescript
    ```
  </Step>
  
  <Step title="Compile the project">
    ```bash
    tsc
    ```
    
    This compiles all `.ts` files from `src/` to JavaScript in the `dist/` directory.
  </Step>
</Steps>

<Note>
  The `tsconfig.json` is already configured with `"rootDir": "./src"` and `"outDir": "./dist"`, so the compiled files will be in the right place.
</Note>

## Run the Simulation

Open `index.html` in a modern web browser:

<CodeGroup>
```bash macOS
open index.html
```

```bash Linux
xdg-open index.html
```

```bash Windows
start index.html
```
</CodeGroup>

Or simply drag the `index.html` file into your browser window.

## What You're Seeing

When you open the simulation, you'll see two celestial bodies:

### The Sun (Orange, Static)

From `src/main.ts:33-39`:

```typescript
const Sun = new Body({
  position: { x: tc.width / 2, y: tc.height / 2 },
  mass: 1000000 * 10000000,
  radius: 250,
  color: 'orange',
  isStatic: true
})
```

<Info>
  The `isStatic: true` flag means the Sun stays in place and doesn't respond to gravitational forces from other bodies.
</Info>

### The Earth (Green, Dynamic)

From `src/main.ts:41-46`:

```typescript
const Earth = new Body({
  position: { x: tc.width / 3, y: tc.height / 3 },
  mass: 1000000 * 100000,
  radius: 45,
  color: 'green'
})
```

The Earth will be pulled toward the Sun by gravitational force. Without an initial velocity, it falls directly toward the center.

## Interactive Controls

The default configuration includes a click handler that adds velocity to the Earth:

```typescript
function controlMove() {
  document.addEventListener('click', ()=>{
    Earth.velocity.x += 0.3
  })
}
```

<Tip>
  **Try this:** Click the canvas several times to give Earth horizontal velocity. With enough speed, it will orbit the Sun instead of falling into it!
</Tip>

## What's Happening Behind the Scenes

Every frame, the simulation:

1. **Calculates forces** - Each body computes gravitational forces from all other bodies
2. **Updates acceleration** - Using Newton's second law: `a = F/m`
3. **Updates velocity** - `velocity += acceleration * dt`
4. **Updates position** - `position += velocity * dt`
5. **Renders** - Draws each body as a colored circle on the canvas

From `src/main.ts:8-27`, the main loop:

```typescript
tc.loop((dt: number) => {
  tc.clear();

  for (let i = 0; i < bodies.length; i++) {
    let body = bodies[i];
    tc.circle(body.position.x, body.position.y, body.radius, body.color)
    body.update(dt, bodies)
  }

  tc.text((bodies[bodies.length - 1]?.speed * 1000 / 36).toFixed(2) + 'cm/s', 140, 140, 70)
  tc.text(tc.fps + "fps", tc.width - (140 + 70), 140, 70)
})
```

<Check>
  The simulation displays real-time FPS in the top-right corner and the current speed of the last body (Earth) in the top-left.
</Check>
